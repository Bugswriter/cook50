Project I wrote for fun

Cook50 is a virtualized, personalizable cookbook. Register to use it.

I got inspired by Tulsi Patel final project at CS50.

Features and Functionalities

  • Create accounts - User can register and then login to start their session.
  • Recipe book - User can add and view recipes.
  • Shopping List - User can add items to shopping list.
  • Meal Planner - User will get a meal calender, where they can add their recipes.

Technologies

  • Flask - Backend is written in python framework flask.
  • Sqlite3 - SQL queries on sqlite3 database is used to make CRUD functions.
  • Ajax - Ajax is used to handle some GET/POST requests.
